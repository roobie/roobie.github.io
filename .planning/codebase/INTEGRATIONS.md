## APIs & External Services

**Custom Analytics:**

## CI/CD & Deployment

**Hosting:**
- GitHub Pages - Free static hosting
- Repository: `roobie/roobie.github.io`
- Site URL: `https://roobie.github.io/`

**CI Pipeline:**
- GitHub Actions workflow: `.github/workflows/deploy.yml`
- Trigger: Push to `master` branch or manual `workflow_dispatch`
- Build process:
  1. Checkout repository (actions/checkout@v5)
  2. Build with `withastro/action@v5`:
     - Install dependencies via pnpm
     - Run `astro check` and `astro build`
     - Generate Pagefind search index: `pagefind --site dist`
     - Copy index to public: `cp -r dist/pagefind public/`
  3. Deploy to GitHub Pages (actions/deploy-pages@v4)
- Permissions: read contents, write pages, write id-token
- Environment: Ubuntu latest
