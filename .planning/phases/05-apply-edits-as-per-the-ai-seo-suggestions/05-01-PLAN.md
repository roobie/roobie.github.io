**Layout.astro** (five changes):

1. **Props type** (line 9-19): Add `tags?: string[];` after the `scrollSmooth` field.

2. **Destructuring** (line 21-31): Add `tags,` to the destructured props (after `scrollSmooth`).

3. **structuredData object** (lines 35-49): Replace the unconditional `structuredData` object with a conditional one. When `pubDatetime` is truthy, build a BlogPosting object that includes `description`, optional `keywords` (from `tags.join(", ")` when tags exist and are non-empty), and uses `pubDatetime.toISOString()` directly (not a template literal). When `pubDatetime` is falsy, set `structuredData` to `null`. Exact pattern from 05-RESEARCH.md Pattern 3:
```
const structuredData = pubDatetime
  ? {
      "@context": "https://schema.org",
      "@type": "BlogPosting",
      headline: title,
      image: `${socialImageURL}`,
      datePublished: pubDatetime.toISOString(),
      ...(modDatetime && { dateModified: modDatetime.toISOString() }),
      description: description,
      ...(tags && tags.length > 0 && { keywords: tags.join(", ") }),
      author: [
        {
          "@type": "Person",
          name: author,
          ...(profile && { url: profile }),
        },
      ],
    }
  : null;
```

4. **og:type meta tag** (after line 85, before the og:image line): Insert `<meta property="og:type" content={pubDatetime ? "article" : "website"} />` between the og:url and og:image meta tags. This follows the recommended OG property ordering: title, description, url, type, image.

5. **article:author and article:tag** (after the existing article:modified_time block, around line 104): Add two conditional blocks:
```
{
  pubDatetime && profile && (
    <meta property="article:author" content={profile} />
  )
}
{
  pubDatetime &&
    tags?.map(tag => (
      <meta property="article:tag" content={tag} />
    ))
}
```

6. **JSON-LD script tag** (lines 113-118): Wrap the existing script tag in a conditional so it only renders when `structuredData` is non-null:
```
{
  structuredData && (
    <script
      type="application/ld+json"
      is:inline
      set:html={JSON.stringify(structuredData)}
    />
  )
}
```
